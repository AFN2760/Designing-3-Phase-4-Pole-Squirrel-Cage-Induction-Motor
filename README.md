# Designing a 3-Phase 4-Pole Squirrel Cage Induction Motor

An academic industrial design engineering report detailing the foundational math, slot-winding geometries, insulation classifications, and stator layouts required to engineer a 3-phase, 4-pole squirrel cage induction motor.

## Core Design Concepts Evaluated
- **Stator Core & Slot Design:** Optimization of slot geometry, copper packing space-factor, and magnetic flux densities across the stator tooth and yoke.
- **Winding Configuration:** Calculation of Standard Wire Gauges (SWG), coil pitches, and group layouts to achieve balanced, symmetrical three-phase sinusoidal air-gap fluxes.
- **Insulation Systems:** Evaluation of Class F thermal insulation properties designed to sustain maximum continuous hotspot limitations up to 155°C.

## References & Design Frameworks
This project was constructed in strict compliance with classic electric machinery guidelines:
- S. J. Chapman, *Electric Machinery Fundamentals*, 5th ed.
- Standard Wire Gauge (SWG) engineering tables for current density optimizations.
